## 🚀 Quick Start

### View Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/[your-username]/[repo-name].git
   cd [repo-name]
   ```

2. Open `index.html` in your browser:
   ```bash
   # On Mac
   open index.html
   
   # On Windows
   start index.html
   
   # On Linux
   xdg-open index.html
   ```

### Deploy to GitHub Pages

1. Push code to GitHub (see deployment instructions below)
2. Go to repository **Settings**
3. Navigate to **Pages** section
4. Under "Source", select branch: **main** (or master)
5. Click **Save**
6. Wait 1-2 minutes
7. Your site will be live! 🎉

## 🔧 Setup Form Submissions

To enable contact forms:

1. Sign up at [Formspree](https://formspree.io) (FREE)
2. Create a new form
3. Copy your form endpoint ID
4. In `index.html`, replace `YOUR_FORM_ID` with your actual ID:
   ```html
   action="https://formspree.io/f/YOUR_ACTUAL_ID"
   ```
5. Forms will now email to: **longfresh05@gmail.com**

## 🎨 Customization

### Change Brand Colors

In `index.html`, find the Tailwind config:

```javascript
colors: {
    'brand-orange': '#FF8C00',  // Primary color
    'brand-yellow': '#FFD700',  // Accent color
    'brand-green': '#4CAF50',   // Success color
}
```
